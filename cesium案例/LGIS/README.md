# LGIS
Based on cesium 3D GIS platform, support WMTS WFS PBF MVT , OSGB……
## 运行简介
该工程基于Vue开发。运行步骤如下
* 1.下载node.js并安装
* 2.在项目路径 运行 npm install，安装需要依赖
* 3.npm run serve，启动项目，默认端口为8080
* 4.在浏览器中输入localhost:8080即可
* 5.打包可以运行 npm run build。
